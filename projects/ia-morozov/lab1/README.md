# Лабораторная работа 1. Развёртывание и эмпирический анализ LLM

## Описание задания

В работе были развернуты три модели из разных семейств(Qwen, Google, Mistral) с помощью vLLM. Для них подготовлены три типа запросов: генерация текста, классификация и суммаризация. Каждая модель тестировалась с базовыми и изменёнными гиперпараметрами (temperature, top_p и repetition_penalty). Сделаны выводы о влиянии параметров и различиях между семействами моделей.


## Использованные технологии и модели

- **LLM**:
  - Qwen/Qwen2.5-7B-Instruct
  - mistralai/Ministral-8B-Instruct-2410
  - google/gemma-3-4b-it
- **Docker**, **vLLM** - для сервинга моделей
- **Python**, **OpenAI** - для тестирования
